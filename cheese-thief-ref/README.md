# 🧀 Cheese Thief — จอมโจรชีส

เกมบอร์ดออนไลน์แนวปาร์ตี้ บลัฟ และจับโกหก สำหรับ 4–8 คน  
เล่นกับเพื่อนได้แบบ Real-Time ผ่านเว็บ

## 🚀 เล่นออนไลน์

> [เปิดเกม](https://your-app-name.onrender.com) ← เปลี่ยนเป็น URL จาก Render.com

## 📦 Run Locally

```bash
npm install
npm start
# เปิด http://localhost:3000
```

## 🛠️ Deploy บน Render.com

1. Push โค้ดขึ้น GitHub
2. ไปที่ [render.com](https://render.com) → New → Web Service
3. Connect GitHub repo นี้
4. ตั้งค่า:
   - **Build Command:** `npm install`
   - **Start Command:** `npm start`
   - **Environment:** Node
5. Deploy!

## 🎮 บทบาท

| บทบาท | จำนวน | เป้าหมาย |
|--------|--------|-----------|
| 🐭 หนูดี | 3–6 คน | จับโจรให้ได้ |
| 🦹 จอมโจรชีส | 1 คน | เนียนให้รอด |
| 😈 ลูกสมุน | 0–1 คน | ช่วยโจรปกปิดความจริง |
